# Directory Traversal Explorer

This project is designed to help security researchers understand and explore directory traversal vulnerabilities. The tool simulates a simple web application that is vulnerable to directory traversal, allowing users to see how such vulnerabilities can be exploited and protected against.

## Features

- Simulates a vulnerable web server that allows directory traversal.
- Provides a safe environment to test and understand directory traversal attacks.
- Includes protection mechanisms to demonstrate mitigation strategies.

## Usage

1. **Clone the Repository**
   ```bash
   git clone https://github.com/your-repo/directory-traversal-explorer.git
   cd directory-traversal-explorer
   ```

2. **Install Dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the Application**
